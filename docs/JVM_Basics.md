# JVM Basics

- **JDK**: Java Development Kit – tools + compiler + JRE.  
- **JRE**: Java Runtime Environment – libraries + JVM to run Java apps.  
- **JVM**: Java Virtual Machine – executes compiled bytecode.

Java source is compiled to **bytecode** (`.class` files).  
Any JVM on any OS can run the same bytecode, which is why Java is *write once, run anywhere*.
